# SkillSync вЂ“ Modern E-Learning Platform

SkillSync is a modern **E-Learning Web Platform UI** built using **HTML, CSS, and JavaScript**.  
The project focuses on creating a clean, responsive, and user-friendly learning interface similar to modern online learning platforms.

The platform allows users to explore courses, read learning resources, view certifications, and interact with a structured learning dashboard.

---

## Features

### Modern UI Design
- Clean SaaS-style interface
- Professional learning dashboard layout
- Smooth navigation system
- Font Awesome icons integration

### Responsive Navigation
- Animated hamburger menu
- Mobile-friendly navigation
- Tablet and desktop optimization
- Smooth transitions

### Multiple Pages
The platform includes several structured pages:

- Home / Explore page
- Blog & Learning Resources
- Certifications page
- Contact page
- Analytics Dashboard
- Authentication (Login / Signup)

### Modular Code Structure
- Organized CSS files
- Separated JavaScript modules
- Clean folder structure
- Easy to scale and maintain

---

## Technologies Used

### Frontend
- HTML5
- CSS3
- JavaScript (Vanilla JS)

### Libraries
- Font Awesome

### Design Approach
- Responsive design
- Modular CSS
- Component-based UI

## How to Run the Project

1. Clone the repository

```
git clone https://github.com/https-jerrick/skillsync.git
```

2. Open the project folder.

3. Run the project by opening:

```
index.html
```

in your browser.

No installation or backend setup is required because the project uses **pure frontend technologies**.
